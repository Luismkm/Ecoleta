## 📋 Sobre

Ecoleta ajuda pessoas a encontrarem pontos de coleta de resíduos de forma rápida e fácil.

----

## 💻 Tecnologias utilizadas

- React
- React Native
- Typescript
- Expo
- Express
- Axios

## ⚙ Como executar

```bash
#clone
$ git clone https://github.com/Luismkm/Ecoleta

# backend
$ cd server
$ npm i
$ npm run knex:migrate
$ npm run knex:seed
$ npm run dev



# web
$ cd web
$ npm i
$ npm start

# mobile
$ cd mobile
$ npm i
$ npm start 

```
